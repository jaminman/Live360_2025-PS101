# Lab 2 — Working With Data (Student Version)

## Goal
Practice filtering, sorting, selecting, and simple data manipulation.

### Task 1 — Filtering
- List all running services
- List all stopped services

### Task 2 — Sorting
- Sort processes by CPU
- Sort services by DisplayName

### Task 3 — Selecting
- Select only Name and Status from services
- Select only Name and CPU from processes

### Task 4 — Calculated Properties
- Create one calculated property of your choice (any object)

### Task 5 — Pipeline Practice
- Build a pipeline with three steps (example: Get → Filter → Sort)

### Task 6 — Object Exploration
- Use Get-Member on any object and identify:
  - 2 methods
  - 3 properties
