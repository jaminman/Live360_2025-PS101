# Lab 3 — Scripting Basics (Student Version)

## Goal
Create simple scripts using variables, loops, logic, and parameters.

### Task 1 — Variables
- Create variables for:
  - A string
  - A number
  - An array

### Task 2 — Loops
- Loop through an array and output each value

### Task 3 — If Statements
- Write a simple if statement based on a numeric variable

### Task 4 — Basic Script
- Create a script that:
  - Accepts a parameter
  - Performs an action (your choice)
  - Outputs something meaningful

### Task 5 — Error Handling
- Add a try/catch block around a failing command

### Task 6 — Multi-Computer Loop
- Create an array of three computer names (fake is fine)
- Loop through them and run one command (ex: Get-Date)
