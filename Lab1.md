# Lab 1 — PowerShell Basics (Student Version)

## Goal
Learn the foundational commands: Get-Help, Get-Command, Get-Member, and basic navigation.

### Task 1 — Using Get-Help
- Explore help for any command
- Run examples
- Write one thing you learned

### Task 2 — Using Get-Command
- List all commands
- Find commands about services
- Find commands using the “Get” verb

### Task 3 — Inspecting Objects
- Run any command
- Pipe it to Get-Member
- Identify three properties

### Task 4 — Exploring Your System
- List processes
- List services
- Sort services alphabetically
- Sort processes by CPU

### Task 5 — Navigating the File System
- Check your current location
- List current directory contents
- Change directories
- Show hidden files
- Return to your original location

### Task 6 — Exploring Providers
- List all providers
- Navigate to a registry path (read-only)
- List keys in that location
- Navigate back

### Task 7 — Discovering Commands
- List available verbs
- Pick a noun (disk, network, computer, etc.)
- Discover commands using that noun
- Inspect one using Get-Help, Get-Command, Get-Member
