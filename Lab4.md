# Lab 4 — Practical Admin Tasks (Student Version)

## Goal
Perform simple administrative tasks using PowerShell.

### Task 1 — Services
- Check the status of a service
- Start or stop a service (safe options preferred)

### Task 2 — Processes
- Find a process by name
- Display selected properties

### Task 3 — File Tasks
- Copy a file
- Create a folder
- List contents recursively

### Task 4 — Logging
- Write a message to a log file
- Append another line

### Task 5 — Modules
- Find a module
- View details about it (without installing)

### Task 6 — Profile
- View your PowerShell profile path
- Add one function or alias
